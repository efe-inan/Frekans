# Frekans

Frekans, zihinleri hizalayan sosyal bir kutu oyunu deneyimidir. İki zıt kavram arasındaki spektrumda gizli hedefin yerini bulmaya çalışırken, takım arkadaşlarınızla aynı dalga boyunda (frekansta) olup olmadığınızı test edin!

## Özellikler

- 🎮 **Oynaması Kolay:** Tarayıcı üzerinden anında oynanabilir.
- 👯 **Takım Modu:** Arkadaşlarınızla iki takıma ayrılıp yarışın.
- 🎯 **Dinamik Kartlar:** Her turda birbirinden farklı, eğlenceli ve düşündürücü zıt kavramlar (Örn: "Sıkıcı - Eğlenceli", "Çok Sıcak - Çok Soğuk").
- 🌈 **Etkileşimli Çark:** Gizli hedefi tahmin etmek için dokunmatik ve fare uyumlu etkileşimli kadran.
- 📱 **Duyarlı Tasarım (Responsive):** Mobil, tablet ve masaüstü cihazların tamamında kusursuz görünüm ve akıcı animasyonlar.
- 🎨 **Modern Arayüz:** Tailwind CSS ve Framer Motion ile güçlendirilmiş, görsel olarak zengin bir kullanıcı deneyimi.

## Nasıl Oynanır?

1. **Kurulum:** Takım isimlerinizi belirleyip oyuna başlayın.
2. **Medyum (İpucu Veren):**
   - Çarkın ortasındaki hedef aralığını sadece siz görebilirsiniz.
   - O turun kavramlarına göre (Örn: Yumuşak - Sert), hedefin bulunduğu konumu yansıtacak kısa ve öz bir ipucu verin.
3. **Takım (Tahmin Edenler):**
   - Medyumun verdiği ipucunu aranızda tartışın.
   - İbreyi, ipucunun spektrumda denk geldiğini düşündüğünüz noktaya sürükleyin ve tahmininizi onaylayın.
4. **Puanlama:** Tahmininiz gizli hedefin merkezine ne kadar yakınsa, o kadar yüksek puan kazanırsınız (Maksimum 4 Puan).

## Kurulum ve Geliştirme

Projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

### Gereksinimler
- Node.js (v18 veya daha güncel bir sürüm)
- npm veya yarn

### Adımlar

1. Depoyu bilgisayarınıza klonlayın:
   ```bash
   git clone https://github.com/kullaniciadi/frekans.git
   cd frekans
   ```

2. Gerekli paketleri yükleyin:
   ```bash
   npm install
   ```

3. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

4. Tarayıcınızda `http://localhost:3000` adresine giderek oyunu görüntüleyin.
